# Changelog

## 0.1.0 (2026-07-07)

- Initial release
- Interactive REPL mode with tab completion
- Setup wizard via `dbh init`
- Docker compose lifecycle: `up`, `down`, `logs`, `restart`
- Status dashboard with accounts, tunnel, POW cache
- Account management: `add`, `list`, `remove`, `test`
- API key management: `add`, `list`, `remove`
- Configuration: `show`, `set`, `edit`
- Diagnostics via `dbh doctor`
- Backup and restore
- Model listing via `dbh models`
- Quick chat via `dbh chat`
- Tunnel management: `status`, `rotate`
- Webhook viewer and test
- GitHub Actions CI
